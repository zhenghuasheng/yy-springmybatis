# yy-springmybatis
spring和mybatis整合实例
入口为自定义main方法，在yt-data .jar，该示例不包含web部分，只包含数据层服务，利用dubbo rpc远程暴露接口

该示例中详细说明了数据源druid的配置，具体可见：https://github.com/alibaba/druid/wiki/DruidDataSource%E9%85%8D%E7%BD%AE%E5%B1%9E%E6%80%A7%E5%88%97%E8%A1%A8

该示例中放置了activeMQ测试代码
